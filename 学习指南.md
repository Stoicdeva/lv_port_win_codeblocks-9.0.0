# LVGL 学习指南

## ? 学习目标

本指南将帮助您从零开始学习 LVGL，配合正点原子教程，掌握嵌入式 GUI 开发技能。

## ? 学习路径

### 第一阶段：环境搭建与基础概念

#### 1. 环境准备
- [x] 安装 CodeBlocks + MinGW
- [x] 克隆 LVGL 模拟器项目
- [x] 成功编译运行第一个程序

#### 2. LVGL 基础概念
- **对象系统**：理解 LVGL 的面向对象设计
- **屏幕管理**：了解屏幕的创建和切换
- **内存管理**：理解 LVGL 的内存分配机制
- **事件系统**：掌握事件处理机制

### 第二阶段：控件学习

#### 1. 基础控件
```c
// 标签控件
lv_obj_t* label = lv_label_create(lv_screen_active());
lv_label_set_text(label, "Hello LVGL!");

// 按钮控件
lv_obj_t* btn = lv_btn_create(lv_screen_active());
lv_obj_set_size(btn, 120, 50);

// 输入框控件
lv_obj_t* textarea = lv_textarea_create(lv_screen_active());
lv_textarea_set_placeholder_text(textarea, "请输入文字...");
```

#### 2. 容器控件
```c
// 容器
lv_obj_t* cont = lv_obj_create(lv_screen_active());
lv_obj_set_size(cont, 200, 150);

// 在容器中创建子控件
lv_obj_t* child_label = lv_label_create(cont);
lv_label_set_text(child_label, "容器中的标签");
```

#### 3. 高级控件
- **滑块（Slider）**：数值输入控件
- **开关（Switch）**：开关状态控件
- **下拉列表（Dropdown）**：选择控件
- **表格（Table）**：数据展示控件

### 第三阶段：样式与主题

#### 1. 样式系统
```c
// 创建样式
static lv_style_t style_btn;
lv_style_init(&style_btn);

// 设置样式属性
lv_style_set_bg_color(&style_btn, lv_color_hex(0x2196F3));
lv_style_set_radius(&style_btn, 10);
lv_style_set_pad_all(&style_btn, 10);

// 应用样式
lv_obj_add_style(btn, &style_btn, 0);
```

#### 2. 主题系统
- 了解内置主题（Default、Monochrome）
- 学习自定义主题创建
- 掌握主题切换机制

### 第四阶段：布局与动画

#### 1. 布局系统
```c
// Flex 布局
lv_obj_set_layout(cont, LV_LAYOUT_FLEX);
lv_obj_set_flex_flow(cont, LV_FLEX_FLOW_COLUMN);
lv_obj_set_flex_align(cont, LV_FLEX_ALIGN_CENTER, LV_FLEX_ALIGN_CENTER, LV_FLEX_ALIGN_CENTER);
```

#### 2. 动画系统
```c
// 创建动画
lv_anim_t a;
lv_anim_init(&a);
lv_anim_set_var(&a, obj);
lv_anim_set_values(&a, 0, 100);
lv_anim_set_time(&a, 1000);
lv_anim_set_exec_cb(&a, set_width);
lv_anim_start(&a);
```

### 第五阶段：实战项目

#### 1. 简单应用开发
- **计算器**：练习按钮和标签的使用
- **设置界面**：练习各种控件的组合
- **数据展示**：练习表格和图表的应用

#### 2. 与硬件结合
- **STM32 移植**：学习 LVGL 在嵌入式平台的应用
- **触摸屏支持**：添加触摸输入功能
- **外部存储**：实现数据持久化

## ? 学习建议

1. **循序渐进**：从简单控件开始，逐步学习复杂功能
2. **多动手实践**：理论结合实践，多写代码多调试
3. **参考示例**：学习官方示例代码，理解最佳实践
4. **记录笔记**：记录学习过程中的重要知识点
5. **参与社区**：加入 LVGL 社区，与他人交流学习

## ? 进阶方向

掌握基础后，可以考虑以下进阶方向：
- **性能优化**：学习 LVGL 性能调优技巧
- **自定义控件**：开发符合项目需求的专用控件
- **多语言支持**：实现国际化功能
- **网络功能**：添加网络通信功能
- **多媒体支持**：集成音频、视频播放功能
